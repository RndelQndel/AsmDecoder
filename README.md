# AsmDecoder

AsmDecoder is a project written in C++.

## Usage

1. Clone this project.
2. cmake -DCMAKE_BUILD_TYPE=[build_feature] .
3. cmake --build .
4. run ./build/[build_feature]/AsmDecoder
